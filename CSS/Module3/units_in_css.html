Unit in HTML

Relative Units

  1. Percentage (%)
    • It is often used to define a size as relative to an element's parent object
    • eg :-
      • width : 33.33% (relative to parent)
      • margin-left : 50% (relative to the parent size)
  2. Em
    • Font size of the parent, in the case of typographical properties like font-size , and font size of the element itself, in the case of the other peoperties like width.
