CSS Transform

This property lets you rotate,scale,skew, or translate an element.

  • Rotate

    • transform : rotate(45deg);

  • Scale

    • transform : scale(0.5);
    • transform : scale(1.5);

  • Translate

    • transform : translate(50px, 50px);
    • transform : translateX(10px);
    • transform : translateY(10px);

  • Skew

    • transform : skew(30deg);
