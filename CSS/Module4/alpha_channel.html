Alpha Channel & Opacity

Alpha Channel

  • Sets the opacity for a color
  • Ranges from 0 to 1
  • Example :-
    • rgba(255,255,255,0.3);

Opacity

  • Sets the opacity for the element
  • Ranges from 0 to 1
  • Example:-
    • opacity : 0.5;
