Forms in HTML

  • Forms are used to collect data from users.
  • <form> tag are used to create from.

  • Action Attribute :

    • Action Attribute is used to define what action needs to be performed when a form is submitted or where the form data should be sent.
    • <from action="/action.php" >
    • <from action="/action" >

  • Form Elements :

    • Input

      • Used to create multiple form controls.
      • There are multiple types of inputs that can be created using type attribute.
      • Type attribute :
        • <input>
        • <input type="text">
        • <input type="password">
        • <input type="number">
        • <input type="time">
        • <input type="color">

    • Placeholder Attribute

      • <input type="text" placeholder="Enter Name ">

    • Label

      • Label Element represents a caption for an item in a user interface.
      • <label>
          Enter your username :
          <input type="text" placeholder="username">
        </label>

    • Button

      • <button> Submit </button>
      • type attribute :
        • <button type="submit"> Submit </button>
        • <button type="button"> Do Something </button>
        • <button type="reset"> Reset </button>
      • Button using Input :
        • <input type="submit" >
        • <input type="submit" value="click me" >

    • name Attribute

      • Name of form control
      • submitted with the form as part of a name/value pair
      • <input type="text" placeholder="enter name" id="username" name="username"/>

    • Checkbox Attribute

      • <input type="checkbox" name="age" id="age" checked />
      • <label for="age"> I am 18+ </label>

    • Radio Button

      • <input type="radio" name="fruit" id="apple" value="apple" />
      • <label for="apple"> Apple </label>

    • Dropdowns

      • <select name="profession" id="profession">
      •   <option vlaue="student"> Student </option>
      •   <option vlaue="developer"> Developer </option>
      • </select>

    • Range

      • <label for="volume"> Volume </label>
      • <input type="range" min="0" max="100" name="volume" />

    • Text Area

      • <label for="feedback"> Please provide your valuable feedback : </label>
      • <textarea name="feedback" id="feedback" cols="30" rows="10"></textarea>
