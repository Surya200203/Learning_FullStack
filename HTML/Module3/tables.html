Tables

  • Tables are used to represent real life data.
  • Tags used in table :-
    • <tr> used to display table row.
    • <td> used to display table column.
    • <th> used to display table header.
    • <thead> used to wrap table header.
    • <tbody> used to wrap table body.
    • <tfoot> used to wrap table footer.
