Image Element

The image element is used to add images to our webpages

  • Image tag is single tag, it only has opening tag not has closing tag.
  • The 'src' attribute in image tag is used to show link of image

      • It also can be of two types :-

    1. Relative link
    2. Absolute link
  • The 'alt' attribute is used to give the alterrnate name for the image
