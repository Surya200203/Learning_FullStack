Algebric Formulas

The below are some mathematical algebri formulas :-

  • a2 - b2 = (a-b)(a+b)
  • (a+b)2 = a2 + 2ab + b2
  • (a-b)2 = a2 - 2ab + b2
  • a3 - b3 = (a-b)(a2 + 2ab + b2)
