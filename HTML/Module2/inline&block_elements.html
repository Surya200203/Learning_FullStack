Block and Inline Elements

Block Elements

  • Takes up the full-width available(whole block)
  • Each block element starts from new line
  • Example :-
    • Heading tag :- <h1>, <h2>, <h3> etc.
    • Paragraph Tag :- <p>

Inline Element

  • Takes up only necessary width
  • Don't start from new line
  • Example :-
    • Anchor tag :- <a>
    • Image Tag :- <img>
