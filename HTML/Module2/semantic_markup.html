Semantic Markup

  • It is the markup that relates to the meaning of content
  • The layout of website becomes more structured.
  • Website becomes SEO(Search Engine Optimization) friendly.
  • Makes our website more readable.

Semantic Tags

The tags mentioned below are some semantic tags used in HTML :-

  • Header Tag :- <header> </header>
  • Main Tag :- <main> </main>
  • Footer Tag :- <footer> </footer>
  • Nav Tag :- <nav> </nav>
  • Article Tag :- <article> </article>
  • Section Tag :- <section> </section>
  • Aside Tag :- <aside> </aside>
