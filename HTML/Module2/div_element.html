Div Element

  • Div is a container used to hold other HTML elements or group of elements together.
  • It is a block element.
  • It is also called as Content Division Element.

    • Code

    <div>
     <a href="www.google.com">Google</a>
     <a href="www.youtube.com">You Tube</a>
    </div>
    <div>
     <a href="www.facebook.com">Facebook</a>
    </div>

    Output

    Google You Tube
    Facebook
